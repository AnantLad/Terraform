# Infrastructure as Code (IaC)

## 📌 What is IaC?
Infrastructure as Code (IaC) is a way to manage and provision infrastructure (servers, networks, etc.) using code instead of manual processes.

---

## ❌ Before IaC (Traditional Way)

Before IaC, managing infrastructure had many problems:

### 🔹 Manual Configuration
- Servers were set up manually  
- High chances of human error  

### 🔹 No Version Control
- Changes were not tracked  
- Difficult to roll back to previous setups  

### 🔹 Documentation Dependency
- Teams relied heavily on written steps  
- Docs often became outdated  

### 🔹 Limited Automation
- Only basic scripts were used  
- Not flexible or scalable  

### 🔹 Slow Provisioning
- Creating new environments took a lot of time  
- Delayed project delivery  

---

## ✅ With IaC (Modern Approach)

IaC solves these problems by:

- Automating infrastructure setup  
- Using code for consistency  
- Enabling version control  
- Making deployments faster and reliable  

---

## 🛠️ Popular IaC Tools

- Terraform  
- AWS CloudFormation  
- Azure Resource Manager  

---

# 🚀 Why Terraform?

## 🌍 Multi-Cloud Support
- Works with AWS, Azure, Google Cloud, etc.  
- Same code can be used across different platforms  

---

## 📦 Large Ecosystem
- Many pre-built modules available  
- Saves time and effort  

---

## 🧾 Declarative Syntax
- You define **what you want**, not how to do it  
- Easy to read and maintain  

---

## 📊 State Management
- Keeps track of current infrastructure  
- Helps detect changes and avoid mistakes  

---

## 🔍 Plan & Apply
- `terraform plan` → Preview changes  
- `terraform apply` → Apply changes  

---

## 👥 Community Support
- Large community  
- Easy to find help and tutorials  

---

## 🔗 Tool Integration
- Works with:
  - Docker  
  - Kubernetes  
  - Ansible  
  - Jenkins  

---

## 🧠 HCL Language
- Uses HashiCorp Configuration Language (HCL)  
- Simple and human-readable  

---

# 💡 Summary

- IaC makes infrastructure **automated, fast, and reliable**  
- Terraform is popular because it's **flexible, powerful, and easy to use**
