## 📌 All Terraform Commands (Command → Usage)

```bash
terraform init                    # Initialize project and download providers
terraform fmt                     # Format Terraform files
terraform validate                # Validate configuration syntax

terraform plan                    # Preview infrastructure changes
terraform apply                   # Apply changes (with approval)
terraform apply -auto-approve     # Apply changes without approval

terraform destroy                 # Delete all managed infrastructure

terraform show                    # Show current state
terraform state list              # List all resources in state
terraform output                  # Show output variables

terraform refresh                 # Sync state with real infrastructure

terraform workspace new dev       # Create new workspace
terraform workspace select dev    # Switch workspace
terraform workspace list          # List all workspaces

```
